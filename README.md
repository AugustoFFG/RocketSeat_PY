# 📒 Agenda de Contatos

Projeto simples em Python de uma agenda de contatos via terminal (CLI), permitindo cadastrar, editar, excluir e favoritar contatos.

---

## 🚀 Funcionalidades

- ✅ Adicionar contatos
- ✏️ Editar contatos existentes
- ❌ Excluir contatos
- ⭐ Favoritar contatos
- 📋 Listar todos os contatos

---

## 🧠 Estrutura dos Contatos

Cada contato é armazenado como um dicionário com os seguintes campos:

```python
{
  "contato": "Nome",
  "telefone": "Telefone",
  "email": "Email",
  "favorito": False
}
